Welcome to App Central!

Application Central is a multi-container usability proof-of-concept that aims to broaden the consumption of network automation via a simplified UX. The project leverages the following technologies:

  • Python3
  • HTML5 / CSS
  • JavaScript
  • Flask / Nginx
  • Netmiko w/ TextFSM
  • Fernet Encryption
  • Docker w/ Compose
  • ServiceNow (ESP)
  • PostgreSQL
  • Redis
  • SparkBot
  • HTML Emails
  • Jinja2 Templates
